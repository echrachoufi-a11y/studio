# **App Name**: Meridian Logistics

## Core Features:

- Service Showcase: Visually highlight maritime, land, and customs management services.
- Quotation Request: Enable clients to submit detailed quotation requests online.
- Client Testimonials: Display authentic client testimonials to build trust and credibility.
- AI-Powered Route Optimization: Suggest optimal routes using AI, analyzing factors like distance, traffic, and customs efficiencies as a planning tool.
- Real-Time Tracking: Provide clients with real-time tracking of their shipments.

## Style Guidelines:

- Primary color: Deep navy blue (#2E4A62) to convey trust and professionalism, echoing maritime themes.
- Background color: Light, desaturated blue-gray (#E5E8EB) for a clean, formal backdrop.
- Accent color: Muted gold (#A89978) to highlight key information and calls to action.
- Headline font: 'Playfair', a modern serif for elegance; body font: 'PT Sans', a modern sans-serif for readability.
- Use minimalist, professional icons representing shipping containers, ships, and trucks.
- Maintain a clean, grid-based layout with generous whitespace to promote clarity and readability.
- Incorporate subtle loading animations and transitions to provide a smooth user experience.
